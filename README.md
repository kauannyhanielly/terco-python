# Terço em Python

Este é um programa em Python que guia a oração do terço passo a passo.

O usuário pode escolher o tipo de mistério:
- Gozosos
- Dolorosos
- Gloriosos
- Luminosos

O sistema conduz as orações com pausas e leitura progressiva.

## Objetivo

Projeto criado para praticar lógica de programação e desenvolvimento em Python.

## Observação

Parte do código foi otimizada com auxílio de inteligência artificial para evitar repetições.
